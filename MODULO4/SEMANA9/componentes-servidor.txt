Sistema operativo:
Es el software principal que administra los recursos del computador o servidor. Permite ejecutar programas y gestionar archivos, memoria y procesos.

Node.js:
Es un entorno de ejecución que permite ejecutar aplicaciones JavaScript fuera del navegador y crear servidores web.

Aplicación server.js:
Es el archivo que contiene el código del servidor y define cómo responder a las solicitudes de los usuarios.

Puerto:
Es un canal de comunicación que permite que el servidor reciba solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre la actividad del servidor, permitiendo monitorear eventos, errores y el funcionamiento general del sistema.
