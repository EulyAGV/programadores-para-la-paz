Al ejecutar el comando node server.js se inicia el servidor Node.js y queda escuchando solicitudes en el puerto 3000.

En la terminal aparece el mensaje:
"Servidor ejecutándose en puerto 3000".

Cuando se visita la ruta http://localhost:3000/estado, el servidor responde con información en formato JSON indicando que el servidor está funcionando correctamente y que el servicio corresponde a una API comunitaria.

Respuesta obtenida:

{
  "estado": "Servidor funcionando",
  "servicio": "API comunitaria"
}
