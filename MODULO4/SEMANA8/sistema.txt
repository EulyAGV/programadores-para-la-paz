El comando uname -a muestra información general del sistema operativo, como el nombre del sistema, versión del kernel, arquitectura y otros detalles técnicos.
El comando whoami muestra el nombre del usuario que actualmente está utilizando el sistema.
Conocer esta información es útil para administrar servidores porque permite identificar el entorno de trabajo, verificar configuraciones del sistema y facilitar tareas de mantenimiento, soporte y seguridad.
