Backlog de la solución comunitaria (ordenado por prioridad)

1. Como ciudadano quiero registrar una incidencia para que quede evidencia y pueda recibir seguimiento.

2. Como coordinador quiero recibir una notificación en Telegram cuando se registre una nueva incidencia para atenderla oportunamente.

3. Como miembro del equipo quiero consultar el listado de reportes para hacer seguimiento a cada caso.

4. Como administrador quiero actualizar el estado de las incidencias para informar el avance a la comunidad.

5. Como ciudadano quiero consultar el estado de mi reporte para conocer si ya fue atendido.
